Hướng dẫn đăng tải tài liệu

  1. Chuẩn bị file (PDF/DOCX/PPTX/ZIP code).
  2. Đặt tên chuẩn: HocPhan_Tuan_Muc-Tieu_PhiênBan (VD: WebCoBan_Tuan03_ThuVienLayout_v1.2.pdf).
  3. Đăng file lên Google Drive/OneDrive và dán link ở từng học liệu.
  4. Cập nhật mô tả ngắn (mục tiêu, yêu cầu, cách chấm).
